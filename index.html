수수의 게임들

넘즐

🎮 게임 소개

넘즐 (NUMZZLE)=넘버(NUMBER) + 퍼즐(PUZZLE)

넘즐은 수학 감각과 창의력을 발휘해야 하는 재미있는 계산 게임입니다.
한 라운드마다 주어진 카드로 수식을 만들고,목표 숫자에 가장 가까운 값을 만든 사람이 점수를 얻습니다.

📋 게임 규칙

  • 목표 숫자를 설정하고 게임을 시작합니다.
  • 숫자 카드와 연산 카드가 랜덤으로 주어집니다.
  • 주어진 카드로 수식을 만들어 목표 숫자에 가장 가까운 값을 제출합니다.
  • 실수했을 경우 초기화 버튼으로 수식을 다시 만들 수 있습니다.
  • 결과값이 서로 다를 경우, 목표 숫자에 가까운 사람이 10점을 받습니다.
  • 총 10라운드를 진행하며, 누적 점수가 더 높은 쪽이 승리합니다!
  • 연습 게임은 컴퓨터와 같은 카드를 받고, 실전 게임은 다른 카드를 받습니다.
  • 게임의 난이도에 따라 넘즐초는 쉬운 버전이고, 넘즐잼은 보통 정도의 난이도이고, 넘즐은 어려운 버전이고, 넘즐업은 계산기가 필요한 매우 어려운 버전입니다.

🎮 Game Overview

NUMZZLE = NUMBER + PUZZLE

Numzzle is a fun and strategic math game that challenges your sense of numbers and creativity.

📋 Game Rules

  • Set the target number and start the game.
  • You receive random number and operator cards.
  • Use number and operator cards to build and submit an expression.
  • Build an expression using the cards to get as close as possible to the target number.
  • If you make a mistake, use the reset button to start your expression over.
  • If the results differ, the one closer to the target number gets 10 points; the other gets 0.
  • After 10 rounds, the player with the higher total score wins.
  • Practice Mode uses the same cards as the computer, while Challenge Mode gives different cards.
  • Numzzle Cho is easy, Numzzle jam is medium, and Numzzle is hard, Numzzle Up is very hard — you might need a calculator!
