넘즐

🎮 게임 소개

넘즐 = 넘버(NUMBER) + 퍼즐(PUZZLE)

📋 게임 규칙

  • 한 라운드마다 주어진 카드로 수식을 만들고, 목표 숫자에 가장 가까운 값을 만든 사람이 점수를 얻습니다.
  • 넘즐초는 덧셈(+)과 뺄셈(-) 연산만 사용합니다. 초등학교 2학년 이상이면 게임할 수 있습니다.
  • 넘즐잼은 덧셈(+), 뺄셈(-), 곱셈(x) 연산을 사용합니다. 초등학교 3학년 이상이면 게임할 수 있습니다.
  • 넘즐은 덧셈(+), 뺄셈(-), 곱셈(x), 나눗셈(÷) 연산기호를 사용합니다. 초등학교 4학년 이상이면 게임할 수 있지만 사고력이 필요한 게임입니다.
  • 넘즐업은 덧셈(+)과 뺄셈(-), 곱셈(×), 나눗셈(÷), 루트(√), 파이(π)를 사용하며 깊은 사고력과 수학적 감각이 필요합니다. 중학교 2학년 이상이면 게임할 수 있습니다.

🎮 Game Overview

NUMZZLE = NUMBER + PUZZLE

📋 Game Rules

  • In each round, create an expression using the given cards, and the player who makes the value closest to the target number earns points.
  • Numzzle Cho uses only addition (+) and subtraction (-) operations. Suitable for students in Grade 2 or above who are comfortable with basic addition and subtraction.
  • Numzzle Jam uses addition (+), subtraction (-), and multiplication (x) operations. Suitable for students in Grade 3 or above who have learned multiplication tables.
  • Numzzle uses addition (+), subtraction (-), multiplication (x), and division (÷) operation symbols. Suitable for students in Grade 4 or above. However, it's a game that requires critical thinking skills.
  • Numzzle Up uses addition (+), subtraction (-), multiplication (×), division (÷), square root (√), and pi (π), requiring deep thinking skills and mathematical sense. Suitable for students in Grade 8 or above.
